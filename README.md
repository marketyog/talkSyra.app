## Installation & Setup

### Requirements
- Any modern web browser
- Internet connection (for downloading fonts and icons)

### How to Use

1. **Extract the files** to your web server or hosting provider
2. **Open `index.html`** in a web browser
3. The website will load with all features ready

### Local Development

1. Clone or download the project files
2. Open `index.html` directly in your browser
3. Or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js (with http-server package)
   npx http-server
   ```
4. Visit `http://localhost:8000` in your browser

## Customization Guide

### To change colors:
Edit `:root` variables in `styles.css`:
```css
:root {
    --primary-color: #ff006e;
    --secondary-color: #8338ec;
    --accent-color: #3a86ff;
    /* ... other colors */
}
```

### To modify contact information:
Edit the Contact section in `index.html`:
- Phone numbers in the contact section
- Email addresses
- Social media links in footer

### To change app download link:
Search for the Play Store link in `index.html` and replace:
```html
https://play.google.com/store/apps/details?id=com.talksyra.app
```
